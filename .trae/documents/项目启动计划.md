# 音乐网站项目启动计划

## 项目概述

这是一个基于 Vue3 + Spring Boot + MySQL 的音乐网站项目，包含三个主要模块：
- **music-client**: 前台客户端 (Vue3 + TypeScript)
- **music-manage**: 后台管理端 (Vue3 + TypeScript)
- **music-server**: 后端服务 (Spring Boot + MyBatis)

## 技术栈

### 后端
- Spring Boot 2.6.2
- MyBatis Plus
- MySQL 8.0
- Redis
- MinIO (文件存储)
- Java 8

### 前端
- Vue 3.2
- TypeScript 4.5
- Element Plus
- Vue Router 4
- Vuex 4
- Axios

## 启动步骤

### 前置条件检查

1. **Java 环境**: JDK 8 或更高版本
2. **Node.js**: 14.17.3 或更高版本
3. **MySQL**: 5.7.21 或更高版本
4. **Redis**: 5.0.8 或更高版本
5. **Maven**: 用于构建后端项目

### 步骤 1: 数据库配置

1. 创建 MySQL 数据库 `tp_music`
2. 导入 SQL 文件：`music-server/sql/tp_music.sql`
3. 检查数据库连接配置：`music-server/src/main/resources/application-dev.properties`
   - 确认用户名和密码正确（默认：root / 123456）

### 步骤 2: 启动 Redis

```bash
redis-server
```

### 步骤 3: 启动后端服务

进入 music-server 目录：

```bash
cd music-server

# 方式一：使用 Maven Wrapper
./mvnw spring-boot:run

# 方式二：使用系统 Maven（如果已安装）
mvn spring-boot:run
```

后端服务将在端口 **8888** 启动

### 步骤 4: 启动前台客户端

进入 music-client 目录：

```bash
cd music-client

# 安装依赖
npm install

# 启动开发服务器
npm run serve
```

前台客户端默认在端口 **8080** 启动

### 步骤 5: 启动后台管理端

进入 music-manage 目录：

```bash
cd music-manage

# 安装依赖
npm install

# 启动开发服务器
npm run serve
```

后台管理端默认在端口 **8081** 启动

## 访问地址

- 前台客户端：http://localhost:8080
- 后台管理端：http://localhost:8081
- 后端 API：http://localhost:8888

## 注意事项

1. **资源文件**: 根据 README 说明，需要下载歌曲和图片资源放到 music-server 目录下
2. **MinIO**: 如果需要文件上传功能，需要启动 MinIO 服务
3. **端口冲突**: 如果端口被占用，可以修改各项目的配置文件
4. **数据库连接**: 确保 MySQL 服务已启动且配置正确

## 常见问题

1. **图片/音乐加载失败**: 确保 img 和 song 目录在正确的位置
2. **数据库连接失败**: 检查 MySQL 服务是否启动，用户名密码是否正确
3. **Redis 连接失败**: 确保 Redis 服务已启动
