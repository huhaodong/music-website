# 权限管理功能说明

## 一、功能概述

权限管理用于配置系统的访问控制，包括：
- 目录（type=0）：页面左侧菜单的一级分类
- 菜单（type=1）：可点击访问的具体页面
- 按钮（type=2）：页面上的操作按钮

## 二、界面布局

```
┌─────────────────────────────────────────────────────────────┐
│ [添加顶级权限]  [权限模板]                                   │
├──────────────────────┬────────────────────────────────────┤
│  权限树（左侧）       │   权限详情（右侧）                     │
│                      │                                     │
│  ▼ 目录1             │   权限名称：xxx                       │
│    ▼ 菜单1           │   权限编码：PERMISSION-001            │
│      - 按钮1         │   权限类型：菜单                       │
│      - 按钮2         │   上级权限：目录1                      │
│    ▼ 菜单2           │   路由路径：/xxx                      │
│      - 按钮3         │   组件路径：xxx.vue                   │
│                      │   排序：1                             │
│                      │   图标：User                          │
│                      │   状态：启用                          │
│                      │   是否缓存：是/否                      │
│                      │   是否显示：显示/隐藏                  │
│                      │   描述：xxx                           │
└──────────────────────┴────────────────────────────────────┘
```

## 三、权限类型说明

| 类型 | 值 | 说明 | 显示位置 |
|------|-----|------|----------|
| 目录 | 0 | 左侧菜单的一级分类（文件夹图标） | 菜单树 |
| 菜单 | 1 | 可访问的页面（页面图标） | 菜单树 |
| 按钮 | 2 | 操作按钮（按钮图标） | 菜单树 |

## 四、使用流程

### 4.1 添加顶级权限

1. 点击「添加顶级权限」按钮
2. 填写权限信息：
   - **权限类型**：选择目录/菜单/按钮
   - **权限名称**：输入显示名称（必填）
   - **路由路径**：type≠2时填写，如 `/user`
   - **组件路径**：type=1时填写，如 `UserManage`
   - **图标**：type≠2时填写，如 `User`
   - **排序**：数字越小越靠前
   - **状态**：启用/禁用
   - **是否显示**：显示/隐藏（在菜单中是否可见）
   - **是否缓存**：type=1时可选，页面是否启用缓存
3. 点击「确定」保存

### 4.2 添加子权限

1. 在权限树中，点击某个权限右侧的「添加」按钮
2. 系统自动填充上级权限信息
3. 填写其他权限信息
4. 点击「确定」保存

### 4.3 编辑权限

1. 在权限树中，点击某个权限右侧的「编辑」按钮
2. 修改权限信息
3. 点击「确定」保存

### 4.4 删除权限

1. 在权限树中，点击某个权限右侧的「删除」按钮
2. 确认删除提示

**注意**：有子权限的权限不允许删除，需先删除所有子权限。

## 五、权限模板

点击「权限模板」可快速创建一组相关权限：

### 5.1 系统管理模板

| 模板 | 包含权限 |
|------|----------|
| 用户管理模板 | 用户管理(目录)、用户列表(菜单)、添加用户、编辑用户、删除用户、分配角色(按钮) |
| 角色管理模板 | 角色管理(目录)、角色列表(菜单)、添加角色、编辑角色、删除角色、分配权限(按钮) |
| 组织管理模板 | 组织管理(目录)、组织列表(菜单)、添加组织、编辑组织、删除组织(按钮) |

### 5.2 内容管理模板

| 模板 | 包含权限 |
|------|----------|
| 音乐管理模板 | 音乐管理(目录)、歌曲列表(菜单)、添加歌曲、编辑歌曲、删除歌曲、上传歌曲(按钮) |
| 歌手管理模板 | 歌手管理(目录)、歌手列表(菜单)、添加歌手、编辑歌手、删除歌手(按钮) |
| 歌单管理模板 | 歌单管理(目录)、歌单列表(菜单)、添加歌单、编辑歌单、删除歌单(按钮) |

### 5.3 使用模板

1. 点击「权限模板」按钮
2. 选择需要的模板
3. 点击「应用模板」按钮
4. 系统自动创建该模板下的所有权限

## 六、权限编码规则

权限编码由系统自动生成，格式为：`PERMISSION-XXX`

例如：`PERMISSION-001`、`PERMISSION-002` 等。

## 七、字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 权限类型 | 是 | 目录/菜单/按钮 |
| 权限名称 | 是 | 显示名称 |
| 上级权限 | 否 | 所属的上级权限 |
| 路由路径 | 否 | 前端路由路径 |
| 组件路径 | 否 | Vue组件路径（菜单类型必需） |
| 图标 | 否 | Element Plus图标名称 |
| 排序 | 否 | 数字越小越靠前，默认0 |
| 状态 | 否 | 启用/禁用 |
| 是否显示 | 否 | 显示/隐藏 |
| 是否缓存 | 否 | 是/否（仅菜单类型） |
| 描述 | 否 | 备注说明 |

## 八、注意事项

1. **权限编码**：由系统自动生成，无需手动输入
2. **删除限制**：有子权限的权限不能直接删除
3. **组件路径**：菜单类型（type=1）建议填写，否则页面无法正常跳转
4. **缓存**：启用页面缓存可提升访问速度，但可能导致数据更新不及时
