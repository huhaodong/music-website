# 音乐网站项目技术文档

## 1. 项目概述

### 1.1 项目简介

本项目是一个基于 **Vue 3 + Spring Boot + MySQL** 开发的全栈音乐网站系统，包含三个主要模块：

1. **music-client（前台客户端）**：面向用户的音乐播放和交互界面
2. **music-manage（后台管理端）**：面向管理员的内容管理系统
3. **music-server（服务端）**：提供 RESTful API 服务的后端系统

### 1.2 项目特点

- 前后端分离架构
- 响应式前端设计
- 完整的音乐播放功能
- 用户认证与授权
- 丰富的社交功能（评论、收藏、评分）
- 支持 Docker 部署

### 1.3 项目结构

```
music-website/
├── music-client/          # 前台客户端（Vue 3）
├── music-manage/          # 后台管理端（Vue 3）
├── music-server/          # 服务端（Spring Boot）
│   ├── sql/               # 数据库脚本
│   └── docker-server/     # Docker 配置
└── doc/                   # 项目文档
```

### 1.4 主要功能列表

#### 前台功能
- 音乐播放（支持歌词同步显示）
- 用户注册/登录（支持邮箱登录）
- 用户信息编辑、头像修改
- 歌曲、歌单搜索
- 歌单打分
- 歌单、歌曲评论
- 歌单列表、歌手列表分页显示
- 音乐收藏、下载
- 播放控制（拖动、音量控制）
- 忘记密码（邮箱验证码重置）

#### 后台功能
- 用户管理
- 歌手管理
- 歌曲管理
- 歌单管理
- 评论管理
- 收藏管理
- 数据统计（ECharts 图表）

## 3. 开发环境要求

- **JDK**: jdk-8u141 或更高版本
- **MySQL**: mysql-5.7.21 或更高版本
- **Redis**: 5.0.8 或更高版本
- **Node.js**: 14.17.3 或更高版本
- **IDE**: IntelliJ IDEA、VSCode
- **MinIO**: 最新版本（可选，用于文件存储）

## 4. 项目启动方式

### 4.1 服务端启动

```bash
cd music-server
mvn spring-boot:run
```

### 4.2 Redis 启动

```bash
redis-server
```

### 4.3 客户端启动

```bash
cd music-client
npm install
npm run serve
```

### 4.4 管理端启动

```bash
cd music-manage
npm install
npm run serve
```

### 4.5 Docker 部署

```bash
docker compose up --build
```
